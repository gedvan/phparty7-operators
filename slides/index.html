Null Coalesce Operator
??

O operador null coalesce ?? retorna o primeiro operando, se ele existir e não for nulo, ou o segundo operando caso contrário

Por exemplo, a expressão:

$a ?? $b

  • Se $a existir e não for NULL, retorna $a
  • Se não, retorna $b

Na verdade, o operador ?? é uma "facilidade sintática" para algumas situações que se repetem com frequência

Por exemplo, quando se deseja testar se um elemento está definido ou não antes de utilizá-lo, de modo a evitar erros

(elementos de arrays, propriedades de objetos, variáveis, etc.)

O operador null coalesce pode ainda ser aninhado:
(com associatividade à direita)


            $valor = $valores['nome'] ?? $outros['nome'] ?? 'Valor padrão';

Spaceship Operator
<=>

O operador spaceship é um novo operador de comparação que funciona da seguinte maneira:

  • Recebe dois operandos
  • Retorna 0 se os operandos forem iguais
  • Retorna -1 se o operando da esquerda for menor que o da direita
  • Retorna 1 se o operando da esquerda for maior que o da direita

Não é tão útil quanto o operator null coalesce, mas o nome é muito melhor :)
